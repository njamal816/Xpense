Xpensly FAQ

FAQ

First Thing First

Xpensly is a forked version of Cashew (https://github.com/jameskokoska/Cashew) which is GPL 3.0 licensed. This app is also GPL 3.0 licensed. You are free to make changes provided that you also share the source code.

Share Data with Partner/Other User

To sync data and share data with multiple users, you need to share a Google account. This ensures privacy when sharing financial data. Xpensly supports syncing of data with a Google account, so a shared Google account will share data between multiple Xpensly apps that use the same account.

Syncing Troubleshooting

If you're experiencing syncing issues between devices, make sure you are logged in to Google Drive with the same account on all devices. Also ensure syncing between devices is enabled. Head to More, Account, login if needed, and tap the Sync/Devices button. Here, Sync Data can be toggled.

Tracking Credit Card Spending

To track credit card spending, create a separate account specifically for tracking credit card spending. You can transfer amounts between accounts to add credit card payments or apply a balance correction. Every time you make a purchase using that credit card, add a transaction to that account.

Transferring Funds Between Accounts

When there is more than one account created within Xpensly, you can transfer funds. There are three primary ways of opening the transfer popup: a new Transfer tab when creating a new transaction, long pressing the (+) button, or editing an account.

Transaction Templates

Xpensly does not offer transaction templates, but you can quickly duplicate common transactions. After long pressing the plus button, you'll see a list of commonly used transactions. Tapping them will duplicate the transaction. If the amount differs, long press on the chip to customize the amount.

Importing CSV / Excel and Data from Other Applications

You can import data in Xpensly by heading to More, Settings & Customization, and scrolling down to "Import & Export". You can import data using a CSV file or by providing a link to a shared Google Sheet.

Using Multiple Currencies

Xpensly only supports one currency per account. You can change the currency by editing an account. However, you can use the currency conversion button when entering an amount to convert between different currencies.

Pro Tips

Customizing Bottom Navigation Bar Pages

To customize the navigation tabs, long press each one to select the page of preference. You cannot change the last tab. Note: You can only customize tabs while in mobile layout.

Long Press the (+) Button

After long pressing the (+) button in the bottom right corner on most pages, you will be presented with more options to 'Add' things apart from the default transaction.

Copy and Paste Amounts

Most amounts within Xpensly can be 'long pressed' to copy the amount. When adding a transaction or entering an amount, you can long press to 'copy' or 'paste' amounts from your clipboard.

Guides

Creating a Budget

To create a budget, first open the Budgets page. Head to More, Budgets, and tap the (+) button. There are two types of budgets: Added Only budget and an All Transactions budget.

Setting Up Category Spending Goals

Once you create a budget, you can set up Category Spending Goals to break down your spending for each category within an overall budget. To add category spending goals, edit the budget and tap Set Category Spending Goals.

Reading a Budget Graph

Xpensly adds useful indicators to help you track your spending trajectory. Within a budget spending breakdown, you will see a line graph showing the amount spent versus date through the budget period.
